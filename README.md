## Quick start

- `git clone https://github.com/timamus/manjaro-minimal-install.git`
- `cd manjaro-minimal-install/`
- `find ./ -name "*.sh" -exec chmod +x {} \;`
- `./1_install.sh`
- `./2_install.sh`
